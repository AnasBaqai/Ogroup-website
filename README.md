# Ogroup Website

The Ogroup Website is a simple website created using HTML, CSS, and Bootstrap 5. It is designed for Ogroup Learning School as a marketing tool to showcase information about the school, its programs, and services. The website is live and hosted on GitHub Pages, providing easy access for users to explore the content.

## Features

- **Responsive Design:** The website is built with a responsive design, ensuring optimal viewing and functionality across various devices and screen sizes.

- **Navigation Menu:** A navigation menu is available to help users easily navigate through different sections of the website.

- **Information Sections:** The website includes sections that provide information about the school, its programs, services, and any other relevant details.

- **Visual Enhancements:** The use of CSS and Bootstrap 5 enhances the visual appeal of the website, making it visually engaging and professional.

## Technologies Used

- HTML: Markup language for structuring the web pages.
- CSS: Stylesheet language for enhancing the appearance of the web pages.
- Bootstrap 5: A popular CSS framework for responsive and mobile-first web development.

## Live Website

The Ogroup Website is live and can be accessed at [https://anasbaqai.github.io/Ogroup-website/](https://anasbaqai.github.io/Ogroup-website/).
